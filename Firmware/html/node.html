NÓ

Info

O Nome do Nó é uma das configurações mais importantes do sistema. Este é utilizado normalmente para identificar o local onde o Onfre está instalado (ex: Cozinha) e tem influência nos seguintes atribuitos:

  • Nome do AP (Access Point) ex: bhonfre-{NOME DO NÓ}-000000
  • DNS LOCAL ex: bhonofre-{NOME DO NÓ}.local | (O DNS Local apenas funciona em sistemas Linux/MAC/BSD)
  • Tópicos MQTT ex: bhonofre/{NOME DO NÓ}/....
